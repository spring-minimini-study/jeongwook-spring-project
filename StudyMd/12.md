# 📚 Spring Framework 학습 기록 (1부: 12장 - RESTful 웹 서비스)

핵심 주제: 화면이 아닌 데이터를 주고받는 방식과 REST 가이드라인의 이해

---

## 1. 현대 웹의 흐름: SPA와 마이크로서비스

과거에는 서버가 화면을 다 만들어서 전달했지만, 이제는 역할이 분리됨

---

### 🧩 SPA (Single Page Application)

- **비유**
  - 메뉴판(HTML)을 한 번 받고, 이후에는 데이터만 계속 받아서 화면을 갱신

- **장점**
  - 페이지 전체 새로고침 없음
  - 사용자 경험이 매우 부드러움

---

### 🏪 마이크로서비스 (Microservices)

- **비유**
  - 하나의 큰 서비스가 아니라 여러 작은 서비스로 분리

- **장점**
  - 장애 격리 가능
  - 기술 선택 자유도 증가

---

## 2. REST란 무엇인가?

REST = 웹 서비스 설계 규칙 (가이드라인)

---

### 📜 REST의 7가지 규칙

- URL로 리소스 식별  
  → /product/p1  

- HTTP 메서드로 행위 표현  
  → GET, POST, PUT, DELETE  

- 응답은 JSON/XML 형식 사용  

- 요청 바디 형식 명시  

- 상태 코드 활용  

- 헤더 활용  

- Stateless  
  → 서버는 상태를 저장하지 않음  

---

## 3. HTTP 메서드

| 메서드 | 역할 | 비유 |
|--------|------|------|
| GET | 조회 | 책 보여줘 |
| POST | 생성 | 새 책 만들어 |
| PUT | 수정 | 책 덮어쓰기 |
| DELETE | 삭제 | 책 버려 |

---

## 4. 상태 코드 (Status Code)

### ✅ 성공 (2xx)
- 200 OK
- 201 Created

---

### ❌ 클라이언트 에러 (4xx)
- 400 Bad Request
- 404 Not Found

---

### 💥 서버 에러 (5xx)
- 500 Internal Server Error

---

## 5. 스프링 MVC의 REST 지원

### 🔄 HttpMessageConverter

- 객체 ↔ JSON 자동 변환
- 일종의 "통번역기"

---

### 🎯 @RestController

- JSON 형태로 데이터 반환
- @Controller + @ResponseBody

---

## 6. 핸들러 메서드 구현 예시
```java
@RestController  
@RequestMapping("/api/trainings")  
public class TrainingRestController {  

    private final TrainingService trainingService;  

    public TrainingRestController(TrainingService trainingService) {  
        this.trainingService = trainingService;  
    }  

    @GetMapping  
    public List<Training> getTrainings() {  
        return trainingService.findAll();  
    }  

    @GetMapping("/{id}")  
    public Training getTraining(@PathVariable String id) {  
        return trainingService.findById(id);  
    }  

    @GetMapping("/info")  
    public String getInfo(@RequestHeader(HttpHeaders.USER_AGENT) String userAgent) {  
        return "당신의 브라우저는: " + userAgent;  
    }  
}  
```
